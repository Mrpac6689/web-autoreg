Instalar Extensão Chrome

Instalação via Arquivo .crx

  1. Clique no botão "Baixar Extensão" abaixo para baixar o arquivo ZIP contendo chrome-extension.crx
  2. Extraia o arquivo ZIP e localize o arquivo chrome-extension.crx
  3. Abra o Chrome e vá para chrome://extensions/
  4. Ative o "Modo do desenvolvedor" no canto superior direito
  5. Arraste e solte o arquivo chrome-extension.crx na página de extensões
  6. Confirme a instalação quando solicitado
  7. Após a instalação, clique no ícone da extensão na barra de ferramentas para configurar:
    • Core do Autoreg: URL onde está rodando o KasmVNC com o core do Autoreg (ex: https://cms.michelpaes.com.br)
    • Frontend: URL da aplicação web Autoreg (ex: https://autoreg.michelpaes.com.br ou http://localhost:5000)

Instalação via Pasta (Modo Desenvolvimento)

  1. Clique no botão "Baixar Extensão" abaixo para baixar o arquivo ZIP
  2. Extraia o arquivo ZIP em uma pasta de sua escolha
  3. Abra o Chrome e vá para chrome://extensions/
  4. Ative o "Modo do desenvolvedor" no canto superior direito
  5. Clique em "Carregar sem compactação"
  6. Selecione a pasta extraída (pasta chrome-extension)
  7. A extensão será carregada e estará ativa
  8. Após a instalação, clique no ícone da extensão na barra de ferramentas para configurar:
    • Core do Autoreg: URL onde está rodando o KasmVNC com o core do Autoreg (ex: https://cms.michelpaes.com.br)
    • Frontend: URL da aplicação web Autoreg (ex: https://autoreg.michelpaes.com.br ou http://localhost:5000)
Nota: A extensão adiciona botões flutuantes na página do KasmVNC para controlar o processo automatizado do Autoreg.
